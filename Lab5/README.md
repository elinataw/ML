### Выводы

В данной лабораторной работе были построены и сравнены несколько моделей на основе деревьев решений для предсказания расхода топлива автомобилей.

1. **Регрессионное дерево** показало среднюю точность на обучающей выборке, однако требует обрезки для уменьшения переобучения. Оптимальное значение параметра `ccp_alpha` подобрано с помощью LOOCV, что позволило немного улучшить качество.
2. **Модель бэггинга** с подобранными параметрами (лучшее число деревьев `n_estimators` и доля выборки `max_samples`, найденные выше) оказалась более точной по большинству метрик на обучающей выборке. Это объясняется тем, что ансамбль усредняет прогнозы множества деревьев, снижая разброс ошибок.
3. При прогнозировании на отложенной выборке лучшая модель показала значение RMSE, приведённое в предыдущем шаге. Это значение следует сравнить с результатами моделей из предыдущего упражнения (если таковые имеются) для окончательной оценки.

Таким образом, **ансамбль бэггинга** с настроенными гиперпараметрами является наиболее точной моделью для данного набора данных и выбранных признаков. LOOCV обеспечивает честную оценку качества, поскольку каждое наблюдение побывало в роли тестового примера.
